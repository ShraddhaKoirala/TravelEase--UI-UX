# 🌍 TravelEase - UI/UX Design

## 📖 Overview
TravelEase is a modern travel booking UI/UX project designed in Figma. It features a clean, user-friendly interface for discovering destinations, browsing travel packages, and managing user authentication through intuitive Login and Sign Up screens.

## ✨ Features
- Modern Hero Section
- Destination Search
- Popular Destinations
- Why Choose Us
- Customer Testimonials
- Responsive Footer
- Login Page
- Sign Up Page
- Interactive Prototype Navigation

## 🛠️ Tools Used
- Figma
- Iconify
- Unsplash
